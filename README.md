# csv-stats
csv-stats — column statistics for CSV files. Reports nulls, unique strings, numeric min/max/avg/median per column.

Part of [supercli](https://github.com/javimosch/supercli) ecosystem.

```bash
curl -LO https://github.com/javimosch/csv-stats/releases/latest/download/csv-stats-linux-amd64
chmod +x csv-stats-linux-amd64; sudo mv csv-stats-linux-amd64 /usr/local/bin/csv-stats
```
